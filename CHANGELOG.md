# Changelog

## v1.0.0 — Borodachamba Music v1.0

- Ретро TUI интерфейс в стиле старых консольных аудио-плееров.
- Кроссплатформенная работа на Windows/Linux (стабильный stop/pause/exit).
- Файловый браузер с добавлением папок/файлов и мультивыбором.
- Плейлист: сохранение/загрузка, удаление треков, очистка.
- Режимы воспроизведения: repeat, shuffle, mute.
- Автовосстановление сессии: плейлист, настройки, последний трек и позиция.
- DSP-цепочка: режимы `stereo/echo/chorus/reverb/phaser/flanger`, уровень DSP.
- Bass Boost (on/off + шаговое усиление).
- EQ пресеты + tone control.
- Ретро VU-индикатор в стиле кассетной деки.
- About/License окно с offline key вводом.
- Скрипты portable сборки для Windows и Linux.
